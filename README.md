# Betting Game

It is web app with Golang backend and Vue frontend.

## Getting Started

### Prerequisites

- [Golang](http://golang.org/)(>11.0)
- [GNU Make](https://www.gnu.org/software/make/)
- [Docker](http://docker.com)

### Environment variables

```bash
	HOST
	PORT
	LOGGER_LEVEL
	DATABASE_URL
```

### Installing and running locally

```bash

#Run server locally 
make run-local

# or

#Run on docker
make run-docker
```
